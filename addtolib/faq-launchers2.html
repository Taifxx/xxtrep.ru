Репозиторий XXTREP включает в себя два аддона:
    
     ATL: Update launcher - для запуска обновления эпизодов
     ATL: Backup launcher - для запуска резервного копирования

Если нужно, Вы можете запустить их, используя Keymap editor (как и сам ADD To Lib) или используя любой лаунчер launcher способный запустить аддон или выполнить код:

Для обновления эпизодов:
RunScript(special://home/addons/context.addtolib/context.py, action=10201)

Для резервного копирования:
RunScript(special://home/addons/context.addtolib/context.py, action=10205)

Другие действия:
RunScript(special://home/addons/context.addtolib/context.py, action=ID)

     ID для других действий:

     30009 - Обновить видео библиотеку
     30027 - Очистить видео библиотеку
     10200 - Сбросить путь библиотеки по умолчанию
     10201 - Запустить обновление эпизодов
     10202 - <только для системы>
     10203 - Выбор цвета меню
     10204 - Запустить переименование источников
     10205 - Запустить резервное копирование
     10206 - Удалить все резервные копии
     10207 - Восстановить библиотеку из резервной копии
     10208 - Сбросить путь резервного копирования по умолчанию
     10209 - Запуск резервного копироваия в автоматическом режиме
     10210 - Выбор скина














Other launcher:

You can use any launcher which runs the add-on or can execute code:
RunAddon(context.addtolib)
